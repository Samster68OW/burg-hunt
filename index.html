About


Burg's Scavenger Hunt was conceptualized in January of 2026 on the Club Penguin Zero Discord. When scavenger hunts are available, players may start one on this website while exploring Club Penguin Island. The objective is to help Burg find all of the objects scattered across the island. Once you find them all, you win the game! You can find the latest scavenger hunts using the dropdown in the top left of the screen.

This site was created using GitHub Pages, Bootstrap 3.4.1, and Glyphicons.
