Burg's Scavenger Hunt


Welcome!

Welcome to Burg's Scavenger Hunt! This is a fun activity to play while waddling around Club Penguin Zero. Burg lost some of Captain Rockhopper's cargo and needs your help to locate it! By using the clues provided, you will need to explore Club Penguin Island and identify the locations of his cargo. You cannot run out of guesses, so have fun!



About


Burg's Scavenger Hunt was conceptualized in January of 2026 on the Zero Nation Discord. When scavenger hunts are available, players may start one on this website while exploring Club Penguin Island. The objective is to help Burg find all of the objects scattered across the island. Once you find them all, you win the game! You can find the latest scavenger hunts using the dropdown in the top left of the screen. This site was created using GitHub Pages, Bootstrap 3.4.1, and Glyphicons. It does not collect any user data.

The latest CPZero version this site recognizes is v1.0.5.5b.

Legacy Scavenger Hunt


On January 21st, 2026, the image below was unofficially released on the Zero Nation Discord server. This hunt was created during the 2026 Winter Party. Burg accidently fell overboard while riding the Migrator to the Winter Party! Can you help him find Rockhopper's cargo that fell over with him? Locate all 10 items on Club Penguin Island to complete the hunt!
